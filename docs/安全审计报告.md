### 🟢 低危：API Key 回退值含占位符

**文件**：`src/config.py` 第 11 行

```python
DEEPSEEK_API_KEY = os.environ.get("DEEPSEEK_API_KEY", "sk-your-key-here")
```

回退值 `"sk-your-key-here"` 是占位符而非空字符串。如果有人部署时忘设环境变量，程序会以假 key 运行，DeepSeek API 调用全部失败，但在 `app.py` 中没有对 API Key 有效性的前置检查——用户可能上传合同后才发现"AI 没反应"。

### 🟢 低危：部分路径硬编码（同陈曦类似模式）

`src/config.py` 第 10-13 行的 embedding 模型 `BAAI/bge-m3` 未验证本地是否已下载。但这一项为非阻塞配置，实际使用 ChromaDB 但当前代码使用纯本地关键词检索，所以无实际风险。
