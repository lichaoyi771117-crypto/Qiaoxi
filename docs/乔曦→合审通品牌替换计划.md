## 三、不替换项（确认保留）

| 类别 | 示例 | 理由 |
|------|------|------|
| AI 人格名「乔曦」 | System Prompt: "你是乔曦（Qiaoxi）" | AI 人设名保留（同文脉通/财析通策略） |
| 模块 docstring | 各文件 docstring | 代码注释，非 UI |
| 私有函数/变量名 | `QiaoxiContractParser`, `QiaoxiLegalReviewer` | 类名 import 绑定 |
| 日志前缀 | `[乔曦初审]` | 日志归档，用户不可见 |
| Prompt 内部引用 | `你 是乔曦` `【乔曦初审摘要】` | LLM 指令，非 UI |
| 环境变量名 | `QIAOXI_ENCRYPTION_KEY` | 配置绑定 |
| 文档文件 | `docs/` 目录 | 开发文档 |
| 测试数据 | `test documents/` 中的报告产物 | 历史产出 |
